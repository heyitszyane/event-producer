# Budget Engine (Budget Manager)

## Purpose
The money source of truth. LLM agents may argue about scope; only this
engine decides what the numbers are.

## Invariants (never weakened)
1. `budget_cap − contingency_reserve − spendable == 0` — contingency is
   reserved before any discretionary spend.
2. `spendable − included_totals − headroom == 0`.
3. Pure function: Decimal-only arithmetic, no I/O, no randomness, no time
   dependence. Same inputs, same summary, forever.

## How it runs
The Budget Manager agent (`agents/budget_manager.py`) gathers inputs from
the event store and calls `compute_budget` directly from code. Scope edits
on the Scope route trigger a recompute through the same path.
