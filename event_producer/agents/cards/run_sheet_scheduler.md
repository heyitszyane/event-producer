# CPM Scheduler (Production Manager)

## Purpose
The time source of truth. Dependencies and lead times are constraints, not
suggestions — if the date is infeasible, the scheduler says so in a conflict
report instead of silently compressing the plan.

## Invariants (never weakened)
1. A task never starts before its dependencies finish.
2. Vendor lead times are respected against the event date.
3. Deterministic: same scope + date always produce the same schedule.

## How it runs
The Production Manager agent (`agents/production_manager.py`) maps scope
items to tasks (duration + dependency catalogue per category) and calls
`compute_schedule` / `derive_call_sheet` directly from code. Scope changes
trigger a recompute with a feasibility report.
